# Futuristic Digital Clock

A highly interactive, futuristic digital clock built with **HTML, CSS, and JavaScript**, featuring a visually engaging UI and real-time dynamic behavior. Perfect for demonstrating modern web design and animation techniques.

- **Random background color changes every second** for a lively interface  
- Animated neon digits with smooth fade/slide effects

- Real-time **day of the week**  
- Current **date in DD/MM/YYYY format**  

### Optional Features
- Hourly chime integration (can be added with a local audio file)  
- Easily extendable to include **weather info, interactive hover effects, or sunrise/sunset themes**
